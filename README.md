# آسو شنو (Aso Shno) - فروشگاه آنلاین و مجله تخصصی لپ‌تاپ 💻

این پروژه یک پلتفرم جامع برای فروش لپ‌تاپ‌های حرفه‌ای و استوک وارداتی، همراه با بخش تعمیرات تخصصی و مجله تکنولوژی است. پروژه با استفاده از جدیدترین تکنولوژی‌های دنیای وب توسعه داده شده است.

## 🚀 تکنولوژی‌های استفاده شده

- **Framework:** [Next.js](https://nextjs.org/) (App Router & Turbopack)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Type Checking)
- **Database:** [PostgreSQL](https://www.postgresql.org/) via [Prisma ORM](https://www.prisma.io/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Authentication:** [NextAuth.js](https://next-auth.js.org/)
- **Storage:** [Liara Object Storage (S3)](https://liara.ir/products/object-storage/)
- **Deployment:** [Liara Cloud](https://liara.ir/)

## ✨ ویژگی‌های کلیدی

- **داشبورد ادمین حرفه‌ای:** مدیریت کامل محصولات، دسته‌بندی‌ها، برندها و مقالات مجله.
- **مدیریت محتوا (CMS):** امکان تغییر لحظه‌ای متون هیرو، بنرها و تنظیمات سئو سایت.
- **ویژگی‌های داینامیک:** سیستم هوشمند افزودن مشخصات فنی متغیر برای هر دسته‌بندی.
- **مجله تخصصی:** بخش مقالات با قابلیت سئو حرفه‌ای و دسته‌بندی.
- **طراحی ریسپانسیو:** تجربه کاربری عالی در موبایل، تبلت و دسکتاپ.
- **فضای ابری S3:** ذخیره‌سازی تصاویر در فضای ابری لیارا برای سرعت و امنیت بالاتر.

## 🛠 راه‌اندازی پروژه (Local)

برای اجرای پروژه در سیستم خود، مراحل زیر را دنبال کنید:

۱. نصب وابستگی‌ها:
```bash
npm install
```

۲. تولید کلاینت Prisma:
```bash
npx prisma generate
```

۳. تنظیم فایل `.env`:
فایل `.env` را بر اساس تنظیمات دیتابیس و فضای ابری خود بسازید.

۴. اجرای پروژه در حالت توسعه:
```bash
npm run dev
```

## 🌐 استقرار در لیارا (Deployment)

این پروژه برای استقرار در پلتفرم لیارا بهینه‌سازی شده است. شما می‌توانید از طریق CLI یا اتصال مستقیم گیت‌هاب به پنل لیارا، فرآیند CI/CD را فعال کنید.
